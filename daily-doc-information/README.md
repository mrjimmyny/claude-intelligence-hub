# daily-doc-information

Automates the creation, structured update, clean-state closure of session documents, and creation of daily executive reports.

## Operations

| Operation | Description |
|---|---|
| `create-session` | Create a new session document from template |
| `update-session` | Add structured content to an open session doc |
| `close-session` | Validate clean-state gate and close a session doc |
| `create-daily-report` | Consolidate session docs into a daily executive report |

## Invocation

```
/daily-doc-information
```

Or ask the agent to create/update/close session docs or create daily reports.

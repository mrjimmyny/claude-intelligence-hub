## What is Conversation Memoria?

Saves VSCode/Abacus.AI Desktop conversations to GitHub with intelligent metadata extraction and token-optimized retrieval.

## Quick Start

**Save:** "Save this conversation"  
**List:** "List conversations from week 7"  
**Search:** "Search conversations about X"  
**Load:** "Load conversation from February 18"

## Key Features

✅ Intelligent metadata extraction  
✅ 95-98% token savings  
✅ Week-based organization  
✅ Natural language triggers  
✅ Cross-agent memory
